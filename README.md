# Uni-n-de-Metodos
Eliminación Gaussiana, bisección y Polinomio de Interpolación con diferencias divididas de Newton
1. Selecciona un metodo a ejecutar.
2. Eliminacion Gaussina: Ingrese el numero de ecuaciones, y posterior ingrese los coeficientes del sistema de ecuaciones (incluyendo termino independiente) separados por espacios.
Ej.
(n): 3
Ecuacion 1: 3 -0.1 -0.2 7.85
Ecuacion 2: 0.1 7 -0.3 -19.3
Ecuacion 3: 0.3 -0.2 10 71.4

3. Biseccion: Ingrese una funcion F(x), despues x1 y xu, para finalizar el error menor esperado en %.
Ej.
f(x): exp(-x) - x
x1: 0
xu: 1
Error esperado: 0

4. Diferencias dividivas: Ingrese la cantidad de n valores, ingrese cada x(n) y y(n) solicitado, y posterior la x a evaluar.
Ej.
Cantidad de valores: 2
x(0): 1
y(0): 7
x(1): 2
y(1): 8
x a evaluar: 2
